# Отчёт по практическому занятию №5
## Выполнил: Лазарев Г.С. 
## Группа: ЭФБО-10-23


### Цели ПЗ

* Отрисовать интерфейс **один в один** по макету (PDF).
* Закрепить работу со **страницами/навигацией** (`Navigator.push / pop / pushNamed`).
* Научиться строить списки через **`ListView.builder`**.
* Реализовать добавление элементов и простое состояние без БД.

---

## Что реализовано

* Экран **входа**: панель с розовой обводкой, кастомная иконка «блокнота», кнопка «Войти» (градиентная «пилюля»).
* Экран **«Задачи»**:

  * заголовок по центру, **круглые чекбоксы** (кастомный виджет),
  * список пунктов «Пункт 1…5» через `ListView.separated`,
  * **квадратный FAB** с «+» в правом нижнем углу.
* Экран **«Новая задача»**: фиолетовый AppBar, большое поле «Текст» с белым контуром, кнопка **«Готово»** (Outlined).
* Единая **палитра** из макета (тёмный фон `#241E35`, фиолетовый AppBar, розовые контуры/кнопки).
* Простая модель `Task` (id, title, done) без хранения на диске.

---

## Технологии

* **Flutter** (Material 3, `ThemeData`).
* Навигация: **named routes**.
* Кастомные UI-элементы: **`CustomPaint`** (иконка блокнота), **кастомный чекбокс** (круг с контуром/заливкой).

## Ключевые экраны

### 1) Вход

* Карточка с **розовой рамкой** и **скруглениями**.
* Кастомная иконка **NotebookIcon** (одинарный блокнот с 3 пружинами).
* Кнопка **«Войти»**: градиент `pink → pinkLight`, скругление 28.

### 2) Список «Задачи»

* Тёмный фон `#241E35`, AppBar фиолетовый, заголовок по центру.
* **CircleCheckbox** слева от текста.
* Отступы/межстрочные как в макете.
* FAB — **квадрат** 64×64, скругление 16, плюс по центру.

### 3) «Новая задача»

* Поле **с белым контуром** (толщина 2), скругление 16.
* Кнопка **«Готово»** — `OutlinedButton` (белый контур, скругление 16), закреплена справа снизу.
